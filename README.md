# Week 1 - Travel Path

Before learning to build AI tools, first understand the world behind it.

This week gives you a big-picture view of what AI is, how it works, where it's used — and how you can be part of it.

✅ What You’ll Learn:

- What exactly is Artificial Intelligence? (not just robots!)
- How the “artificial brain” is trained to think like humans
- Why Python is the heart of AI, and how tools like Jupyter, Anaconda help
- Know AI  – NLP? Data Science? Deep Learning?
- How people earn using AI – jobs, freelancing, startups
- Real vs Myth – AI vs Human brain, Traditional coding vs AI

✅ After This Week:

- You’ll stop fearing tech terms — and start seeing possibilities
- You’ll get a crystal-clear understanding of AI, without confusion
- You’ll be ready to start your AI path
- You’ll understand how AI tools connect and where Python fits in
- Your mindset will shift from "boring learning method" to impactful learning method
- You’ll feel more confident, curious, and future-ready
